- **INSTALLATION.md** - 详细安装说明
- **README.md** - 项目功能介绍

## 🔧 快速安装步骤

### 方法一：使用打包文件（推荐）

1. **解压安装包**
   ```bash
   unzip SuperTab-Chrome-Extension-v1.0.0.zip -d supertab-extension
   ```

2. **Chrome扩展管理**
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹 `supertab-extension`
   - 扩展安装完成！

### 方法二：直接加载文件夹

1. **解压文件**
   ```bash
   unzip SuperTab-Chrome-Extension-v1.0.0.zip -d supertab
   ```

2. **在Chrome中加载**
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹

## 🔍 验证安装成功

1. 安装完成后，在扩展管理页面会显示"SuperTab"扩展
2. 点击扩展图标或侧边栏按钮打开SuperTab
3. 应该能看到当前所有标签页的智能分组显示

## ❓ 常见问题

### Q: 安装后扩展不显示？
A: 确保已开启"开发者模式"，并刷新扩展管理页面

### Q: 侧边栏不显示内容？
A: 尝试刷新当前网页，或重新加载扩展

### Q: 如何卸载扩展？
A: 在 `chrome://extensions/` 页面找到SuperTab，点击"移除"
