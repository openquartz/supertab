## 🔧 安装方法

### 方法一：离线安装 (推荐)

1. **下载安装包**
   ```bash
   # 从GitHub下载
   curl -L -o supertab-latest.zip https://github.com/openquartz/supertab/raw/main/dist/supertab-v1.0.0-latest.zip
   ```

2. **解压文件**
   ```bash
   unzip supertab-latest.zip -d supertab-extension
   ```

3. **Chrome加载**
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹

### 方法二：使用安装脚本

```bash
# 下载安装脚本
curl -O https://raw.githubusercontent.com/openquartz/supertab/main/install.sh
chmod +x install.sh
./install.sh
```
