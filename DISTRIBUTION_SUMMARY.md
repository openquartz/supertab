## 🔧 安装方式

### 方法一：一键安装脚本（推荐）

**Windows用户：**
```bash
双击运行 install.bat
```

**macOS/Linux用户：**
```bash
chmod +x install.sh
./install.sh
```

### 方法二：手动安装

1. 解压 `SuperTab-Chrome-Extension-v1.0.0.zip`
2. 打开Chrome访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的文件夹
