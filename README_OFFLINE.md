# 🎯 SuperTab Chrome Extension - 离线安装包

## 📦 包含文件

```
supertab/
├── SuperTab-Chrome-Extension-v1.0.0.zip    # Chrome扩展安装包
├── install.bat                              # Windows安装脚本
├── install.sh                               # macOS/Linux安装脚本
├── INSTALLATION_GUIDE.md                    # 详细安装指南
├── README.md                                # 完整项目文档
└── RELEASE.md                               # 版本发布说明
```

## 🚀 快速开始

### Windows 用户
1. 双击运行 `install.bat`
2. 按照脚本提示操作
3. 在Chrome中加载扩展

### macOS/Linux 用户
1. 终端运行 `./install.sh`
2. 按照脚本提示操作
3. 在Chrome中加载扩展

### 手动安装
1. 解压 `SuperTab-Chrome-Extension-v1.0.0.zip`
2. 打开Chrome访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的文件夹

## ✨ 核心功能

- 🧠 **智能分组**: 按域名、时间自动分组标签页
- 📝 **标签页笔记**: 为每个标签页添加个性化备注
- 🔍 **快速搜索**: 通过标题、URL、笔记搜索标签页
- ⚡ **批量操作**: 多选标签页进行批量管理
- 🎨 **现代化UI**: 毛玻璃设计，支持深色模式
- 🔒 **隐私保护**: 本地AES加密，无网络传输

## 📊 技术规格

- **版本**: v1.0.0

安装完成后，您可以通过以下方式使用SuperTab：
