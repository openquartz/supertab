# 🚀 SuperTab Chrome Extension - 离线安装指南

## 📦 安装包信息

- **扩展名称**: SuperTab
- **版本**: 1.0.0
- **文件**: `supertab-v1.0.0.zip`
- **大小**: ~2.5 MB
- **兼容性**: Chrome 88+ 或 Chromium 88+

## 🔧 安装步骤

### 方法一：开发者模式安装（推荐）

1. **解压安装包**
   ```bash
   # 解压到指定目录
   unzip supertab-v1.0.0.zip -d supertab-extension
   ```

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或点击Chrome菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关
   - 点击启用开发者模式

4. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 在文件选择器中，选择刚才解压的`supertab-extension`文件夹
   - 点击"选择"或"打开"

5. **确认安装**
   - SuperTab图标会出现在Chrome工具栏中
   - 扩展会自动启用并准备使用

### 方法二：通过Chrome应用商店（即将上线）

> ⚠️ 注意：目前SuperTab正在审核中，即将上架Chrome Web Store

1. 访问 [Chrome Web Store](#)
2. 搜索"SuperTab"
3. 点击"添加至Chrome"
4. 确认权限后完成安装

## 🎯 首次使用

1. **打开SuperTab**
   - 点击Chrome工具栏中的SuperTab图标
   - 或使用快捷键 `Ctrl+Shift+S`

2. **授权权限**
   - 首次使用会请求必要的权限
   - 点击"允许"以启用所有功能

3. **开始使用**
   - 查看自动分组的标签页
   - 点击编辑图标为标签页添加笔记
   - 使用拖拽功能重新组织标签页

## ⚙️ 权限说明

SuperTab需要以下权限来提供完整功能：

| 权限 | 用途 | 必要性 |
|------|------|--------|
| `tabs` | 读取和管理浏览器标签页 | 必需 |
| `storage` | 保存设置和笔记数据 | 必需 |
| `sidePanel` | 显示侧边栏界面 | 必需 |
| `contextMenus` | 右键菜单集成 | 可选 |
| `bookmarks` | 与书签功能集成 | 可选 |

## 🔒 隐私和安全

- 🌐 **无需网络**：完全离线运行，保护隐私

## ❓ 常见问题

**Q: 安装后图标没有出现？**
A: 点击Chrome工具栏中的拼图图标，将SuperTab固定到工具栏。

**Q: 提示"扩展程序已损坏"？**
A: 确保使用完整的安装包，重新解压并加载扩展。

**Q: 如何更新到新版本？**
A: 重复安装步骤，Chrome会自动更新扩展。

**Q: 是否支持其他浏览器？**
A: 目前仅支持Chrome和Chromium内核浏览器（如Edge、Brave等）。

## 🔧 故障排除

### 扩展无法加载
1. 确保Chrome版本 >= 88
2. 检查是否启用了开发者模式
3. 重新解压安装包并重新加载

### 功能异常
1. 刷新扩展页面 `chrome://extensions/`
2. 点击SuperTab的"重新加载"按钮
3. 重启Chrome浏览器

### 数据丢失
1. 检查Chrome的存储权限设置
2. 确保没有启用"隐私模式"
3. 联系开发者获取技术支持
